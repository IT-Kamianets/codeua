Список команд:

  • Друк (print) - друкування сторінки.
  • Якщо (if) - щоб вказати блок коду, який має бути виконано, якщо вказана умова є true (вірна).
  • Функція (function) - це блок коду, призначений для виконання певного завдання.
  • Поки (while) - цей цикл виконає блок коду один раз, перед тим, як перевіряти чи правильна умова, потім він буде повторювати цикл, поки умова істинна (true).
